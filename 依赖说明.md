## 🎯 QMD 是什么？

### QMD = OpenClaw 内置的记忆后端

**不需要单独安装，它是 OpenClaw 的一部分！**

```json
// 在 openclaw.json 中默认配置：
{
  "memory": {
    "backend": "qmd",      // ← 内置的
    "citations": "auto"     // ← 自动检索记忆
  }
}
```

**工作原理：**
1. 当你说"记住：XXX"时，自动写入 QMD
2. 每次对话开始时，QMD 自动检索相关记忆
3. AI 看到相关记忆，自动应用

## 💡 常见问题

### Q1：我需要安装 QMD 吗？

**A：不需要！** QMD 是 OpenClaw 内置的，你安装 OpenClaw 时就有了。

### Q2：我需要安装 Memos 吗？

**A：不需要！** Memos 是完全可选的插件。本 Skill 主要依赖 QMD，不依赖 Memos。

### Q3：为什么我看到文档提到 Memos？

**A：** 因为 Skill **支持** Memos（如果你有的话），但不是**必需**的。Skill 可以完全不需要 Memos 而正常工作。

### Q4：如何确认 QMD 正常工作？

**A：** 运行这个测试：

```bash
# 在 OpenClaw 对话中
/new
记住：我喜欢简洁的回复

# 然后新对话
/new
帮我总结一下快速排序算法
```

**如果 QMD 正常：**
- AI 应该记住你喜欢简洁的回复
- 回复应该比较简短

### Q5：Skill 使用了哪些工具？

**A：** Skill 主要使用 OpenClaw 内置的工具：

| 工具 | 用途 | 是否内置 |
|------|------|:-------:|
| **QMD** | 长期记忆存储和检索 | ✅ 内置 |
| **Read** | 读取配置文件 | ✅ 内置 |
| **Write** | 写入配置文件 | ✅ 内置 |
| **Glob** | 查找文件 | ✅ 内置 |

**所有工具都是 OpenClaw 内置的！**
