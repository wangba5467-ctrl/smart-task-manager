# 快速安装指南

**智能任务管理器 - 3 分钟快速安装**

---

## 🚀 快速安装（所有平台）

### Windows 用户

```powershell
# 1. 复制 skill 目录
Copy-Item -Recurse "smart-task-manager" "$env:USERPROFILE\.openclaw\skills\"

# 2. 运行安装脚本
cd "$env:USERPROFILE\.openclaw\skills\smart-task-manager\bin"
node setup.js

# 3. 重启网关
openclaw gateway restart

# 4. 完成！
```

### macOS/Linux 用户

```bash
# 1. 复制 skill 目录
cp -r smart-task-manager ~/.openclaw/skills/

# 2. 运行安装脚本
cd ~/.openclaw/skills/smart-task-manager/bin
node setup.js

# 3. 重启网关
openclaw gateway restart

# 4. 完成！
```
