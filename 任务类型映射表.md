## 📝 如何添加新流程

### 步骤 1：创建流程文档

```bash
# 在 workspace 创建流程文档
# 路径：~/.openclaw/workspace/新任务流程.md
```

### 步骤 2：更新映射表

在上面表格中添加一行：

```markdown
| **新任务** | 触发词 | `新任务流程.md` | ✅ 已有 |
```

### 步骤 3：测试

```
你：帮我[执行新任务]
AI：应该自动读取新任务流程.md 并执行
```
